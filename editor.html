Line editor

Click a line to select. Drag a dot to move it. Click the line between dots to add one.
Shift-drag a box to select many dots (or Shift-click dots); drag any one to move them together. Delete (or right-click) removes the selected dots. Esc clears.
With dots selected, Straighten snaps them to a line between the ends; Smooth rounds out jitter (click again for more).
Edits auto-save in this browser as you go (survives reloads). Export downloads route-overrides.geojson to commit into data/ for the durable save. Clear local discards un-exported local edits.
