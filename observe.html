each dot = one timepoint departure, colored by route. left→right = minutes late (0 = on time; no early buses). ↑ above = predicted (upcoming buses' current prediction) · ↓ below = measured (departed buses, deviation seen at/after scheduled time). column height = how many departures at that lateness. read the gap: if predicted piles at on-time but measured pushes right, the schedule is optimistic, buses run later than predicted. caveat: measured skews late + sparse (on-time buses vanish before we can catch them, late ones linger), so trust the shape, not the exact %.
